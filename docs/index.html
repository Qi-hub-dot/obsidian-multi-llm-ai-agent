✨ 核心功能

🧠

Agentic 工具调用

AI 自主调用 9 个内置工具完成知识管理操作：搜索笔记、创建笔记、追加内容、浏览目录、管理标签、生成知识图谱。

📷

多模态文件识别

统一附件入口，自动路由：图片 → 视觉模型 OCR、PDF → 渲染提取、Word → mammoth 解析、Markdown → 直接注入。

🔍

Vault 语义检索 (RAG)

本地 TF-IDF 全文索引 + CJK n-gram 分词，零外部依赖。每次对话自动注入 Top-5 相关笔记上下文。

🔄

流式推理 + 思考面板

逐 token 实时输出，DeepSeek V4 Pro 推理链可折叠展示，支持中途停止生成。

🗺️

知识图谱生成

一键生成 Obsidian Canvas 格式脑图，自动布局节点和连线，支持多候选提取 + 格式规范化。

📝

智能笔记创建

融合 Zettelkasten 原子化 + PARA 分类 + MOC 索引方法论，自动归类目录、生成 frontmatter 和标签。

🛠️ 内置工具

工具功能触发场景
searchVault全文搜索笔记「有没有关于X的笔记」
readNote读取笔记全文「打开X笔记看看」
createNote创建新笔记「做笔记 / 创建一篇关于X的笔记」
modifyNote覆盖笔记内容「修改 / 重写 X 笔记」
appendNote追加到笔记末尾「补充 / 追加到 X 笔记」
listNotes列出所有笔记「有哪些笔记 / 看看我的笔记」
getFileTree浏览目录结构「看看我的文件夹结构」
getTags查看标签体系「有哪些标签」
saveCanvas生成知识图谱「画脑图 / 知识图谱 / 思维导图」

📦 安装

方式一：社区插件市场

# Obsidian → 设置 → 第三方插件 → 浏览
# 搜索 "multi-llm-ai-agent"
# 点击安装并启用

方式二：手动安装

# 克隆仓库到插件目录
cd .obsidian/plugins
git clone https://github.com/Qi-hub-dot/obsidian-ai-assistant.git

# 安装依赖并构建
cd obsidian-ai-assistant
npm install && npm run build

# 在 Obsidian 中启用插件
